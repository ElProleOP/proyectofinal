Este proyecto es una pagina web de una Libreria donde el usuario puede registrar libros, juegos de mesa,
hacerse socio o buscarse como socio a traves de un email asociado, registarte como usuario, login , avatar.

1/ La clase Libros consta de 3 caracteristicas: genero, titulo, sinopsis ,un numero que sirve de guia. 
Para acceder a la seccion de registrar un libro se clickea en la barra de arriba (Agregar Libros) 

2/La seccion Libros, sirve para ver en lista los libros de la base de datos, ver los detalles o eliminarlo
Para acceder a la seccion de Libros se clickea en la barra de arriba(Libros)

3/ La clase Juegomesa consta de 3 caracteristicas: titulo, precio y un numero que sirve de guia.
Para acceder a la seccion de registrar un juego de mesa se clickea en la barra de arriba (Juego de mesa) 

4/ La clase Socio consta de 4 caracteristicas: nombre, apellido, edad y un email. 
Para acceder a la seccion de registrarse como socio se clickea en la barra de arriba (Socio)

5/La seccion Registar, sirve para registrarse como usuario
Para acceder a la seccion de REgistrar se clickea en la barra de arriba(Registar)

6/Una vez registrado, puede hacer Login con tu nombre de usuario y contraseña
Para acceder a la seccion de Login se clickea en la barra de arriba(Ingresar)

7/Como usuario que se ha logueado, puedes editar tus datos en la seccion editar perfil
Para acceder a la seccion de Editar Perfil se clickea en la barra de arriba(Editar Perfil)

8/ Por ultimo esta la seccion Buscar Socio, que sirve para buscar los datos de un socio en la base de datos
asociado al email.
Para acceder a la seccion de Buscar socio se clickea en la barra de arriba(Buscar Socio)
